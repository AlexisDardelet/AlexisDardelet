### 📌 Featured projects

#### 🏐 [DataBeach](https://github.com/AlexisDardelet/DataBeach) — beach-volley data & BI platform

An end-to-end project that **creates data where none exists**: from a raw match video, it produces a structured dataset and delivers it as dashboards usable by coaches and players.

- **Full ETL pipeline**: video segmentation → point indexing & score validation → action grading → SQLite database → Firestore migration.
- **Proprietary video-analysis pipeline** (OpenCV + GPU FFmpeg/NVENC): frame-accurate cutting, automated montages, assisted grading.
- **Business-facing BI interfaces** (Streamlit + Plotly), containerised (**Docker**) and deployed on **Google Cloud Run** — accessible anywhere.
- Operational end-to-end, under continuous improvement.

`Python` · `OpenCV` · `FFmpeg` · `SQL` · `Streamlit` · `Firestore` · `Docker` · `GCP`

#### 🤝 [MyVolley](https://myvolley.app/) — scouting & performance-analysis tools

Collaboration with the French platform **MyVolley** on sports performance (scouting) and tournament-management tools:

- **Scouting data sharing** (sports performance) from my analysis pipeline.
- **Co-development of scouting modules**: data visualisation and **machine learning** (automated detection and segmentation).
- **Domain expertise** on scouting and tournament-management tools.

`Machine Learning` · `Computer Vision` · `Data Visualisation` · `Scouting`
